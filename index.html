How it works

  1. Load Rules to bring in your saved KRAs & rules (or add KRAs manually on the right).
  2. Import Timesheet — your HaloPSA CSV; confirm the week's Monday date when asked.
  3. Drag a ticket from the left into a KRA to file it and create a rule for future imports. Click a KRA to expand it.
  4. “Quick Time” entries file themselves automatically.
  5. Dashboard (bottom-right) — hours per KRA by day / week / month / quarter / year, re-orderable, with XLSX export.
  6. Export Rules — the button highlights when you have unsaved changes; save over your rules file to keep them.
